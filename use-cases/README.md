# Use Cases

Real product use cases showing how engineers use Istari day-to-day. Each includes a K-script describing the user experience and a working notebook that achieves the outcome.

| Use Case | Inner Loops | Tools | Question It Answers |
|----------|-------------|-------|-------------------|
| [Understand My Requirements & Architecture](explore-sysml-model/) | 1 | SysGit | See what's inside a systems model — without being a SysML expert |
| [Test a New Wing Shape](run-ntop-wing-design/) | 1 | nTop | Change wing parameters and see how it performs — without spinning up the nTop GUI |
| [Does My Design Meet Requirements?](check-design-meets-requirements/) | 3 | SysGit + nTop | Automatically check that requirements, architecture, and CAD all line up |
