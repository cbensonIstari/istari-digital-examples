# Use Cases

Real product use cases showing how engineers use Istari day-to-day. Each includes a K-script describing the user experience and a working notebook that achieves the outcome.

| Use Case | Inner Loops | Tools | Question It Answers |
|----------|-------------|-------|-------------------|
| [Understand My Requirements & Architecture](explore-sysml-model/) | 1 | SysGit | See what's inside a systems model — without being a SysML expert |
| [Test a New Wing Shape](run-ntop-wing-design/) | 1 | nTop | Change wing parameters and see how it performs — without spinning up the nTop GUI |
| [Does My Design Meet Requirements?](check-design-meets-requirements/) | 3 | SysGit + nTop | Automatically check that requirements, architecture, and CAD all line up |
| [Extract a CATIA V5 Part](extract-catia-v5-part/) | 1 | CATIA V5 | Get parameters, mass properties, and rendered views — without opening CATIA |
| [Extract a NASTRAN FEM Model](extract-nastran-model/) | 1 | NASTRAN Extractor | Get structured mesh, material, and load data — without opening NASTRAN or Patran |
| [Extract a Creo Part](extract-creo-part/) | 1 | Creo Parametric | Get parameters, mass properties, and rendered views — without opening Creo |
| [Extract a Cameo Model](extract-cameo-model/) | 1 | Cameo Enterprise Architect | Get blocks, requirements, and all diagrams — without opening Cameo |
| [Extract a PDF](extract-pdf/) | 1 | PDF Extractor | Get structured text, sections, and semantic chunks — without manual copy-paste |
| [Extract an Excel Spreadsheet](extract-excel-spreadsheet/) | 1 | Microsoft Excel | Get sheet data as CSV, PDF render, and chart data — without opening Excel |
| [Extract a Word Document](extract-word-document/) | 1 | Microsoft Word | Get text, images, and tables — without opening Word |
| [Extract a PowerPoint](extract-powerpoint/) | 1 | Microsoft PowerPoint | Get individual slides, text, and PDF — without opening PowerPoint |
